# Dotfiles

![Starts in 2024](https://img.shields.io/badge/Started-2024-brightgreen)

This repository is used to manage and track my personal configuration files,
managed with [chezmoi](https://github.com/twpayne/chezmoi).
By managing my dotfiles, I can easily set up a new development environment on any machine.

## Features

- Designed to work on macOS.
- Use Homebrew(macOS) for package management.
- Use Zsh as default shell, manage Zsh plugin by [zinit](https://github.com/zdharma-continuum/zinit).
- Customizable prompt by [starship](https://github.com/starship/starship).
- Git configuration and ignore files, enable git signing using SSH Key.
- Manage all runtime versions with [asdf](https://asdf-vm.com/).
